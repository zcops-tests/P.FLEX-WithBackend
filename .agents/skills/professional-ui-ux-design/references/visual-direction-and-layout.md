# Visual Direction and Layout

Use these heuristics when shaping the aesthetic and composition of a screen.

## 1. Pick a mood deliberately

Common directions:

- premium: restrained palette, higher contrast, stronger typography, generous whitespace
- operational: dense but structured layout, fast scanning, strong grouping, status clarity
- friendly: softer edges, approachable copy, lighter rhythm, warm accents
- editorial: large type, bold composition, content-led sections, fewer chrome elements
- technical: crisp alignment, disciplined spacing, clean data presentation, controlled motion

Choose one main direction and avoid mixing incompatible signals.

## 2. Build the page around one dominant focal point

- Put the most important action or information where the eye lands first.
- Use secondary and tertiary emphasis intentionally.
- Reduce nearby noise around the focal point.

## 3. Use spacing as a design tool

- Increase spacing before adding more lines, borders, or containers.
- Let section rhythm communicate structure.
- Keep internal component spacing consistent.

## 4. Control density

- Dense interfaces can still feel premium if grouping and hierarchy are strong.
- Sparse interfaces can still feel weak if there is no focal point.
- Match density to task complexity and user frequency.

## 5. Prefer few strong accents

- Use accent color, shadow, gradients, or illustrations sparingly.
- Let one or two signature moves define the screen.
- Avoid stacking every modern design trope in the same layout.

## 6. Handle dashboards carefully

- Do not default to a grid of identical cards unless the data truly deserves it.
- Create hierarchy between summary, trends, and detailed actions.
- Make the dashboard feel like a tool, not a wallpaper of metrics.

## 7. Handle landing pages carefully

- Lead with a strong promise and clear CTA.
- Use section transitions to build narrative momentum.
- Keep social proof, features, and pricing in a deliberate order.
